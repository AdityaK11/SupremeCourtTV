# SupremeCourtTV

SupremeCourtTV is an innovative project that transforms Supreme Court audio recordings and transcripts into engaging video content. By leveraging advanced AI technologies, this project aims to make Supreme Court proceedings more accessible and visually appealing to the public.

## Features

- **Audio Transcription**: Utilizes Nvidia NeMo ASR for precise transcription of court audio.
- **Transcript Alignment**: Aligns AI-generated transcripts with official court transcripts, providing exact timestamps for each speaker.
- **Video Generation**: Employs Sadtalker to create synchronized videos from images and aligned transcripts.
- **Data Preprocessing**: Implements robust techniques to clean and prepare audio and text data.

## Technologies Used

- Nvidia NeMo ASR for speech recognition
- Sadtalker for video synthesis
- Python for data processing and integration
